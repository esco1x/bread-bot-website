Commands

Moderation

/ban -
Bans a user from a server.

/unban -
Unbans a user from a server.

/kick -
Kicks a user from a server.

/jail -
Jails/mutes a user for a select amount of time.

/jail -
Unjails/unmutes a user.

/warning add -
Warns a user in a server.

/warning remove -
Removes a warning from a user.

/warning check -
Check a users warnings.

/report-setup -
Sets up the report system.

/report-disable -
Disables the report system.

/lock -
Locks a given channel.

/unlock -
Unlocks a given channel.
